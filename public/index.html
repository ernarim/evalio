Evalio Project Information

Project Name: Evalio

Description: This exam database and management application provides modular exam creation and management for universities and institutions. It allows instructors to create exams and tests from a shared, searchable question database that supports multiple courses. The system tracks question usage, supports editing, versioning, and retirement, and offers export options such as LaTeX, Google Docs, and PDF. It provides detailed analytics on question performance to create balanced assessments. The system can also manage assignments and quizzes and offers secure multi-university collaboration with authentication.
